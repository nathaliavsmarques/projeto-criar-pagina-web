HTML e suas Tags

Neste pequeno artigo iremos abordar algumas das tags mais conhecidas e usadas no HTML.

O que é HTML ?

O HTML é uma linguagem de marcação de HiperTexto, responsável por definir a estrutura do conteúdo da web. Para isso ele faz uso de tags semânticas, onde cada uma tem seu significado para estar sendo empregada.

O que são tags

(Voltar)

São estruturas semânticas usadas para que o navegador interprete a estrutura de um site.

< h1 - h6 >

(Voltar)

São usados para determinar o título principal da página e os seus subtítulos. Eles podem se repetir ao longo do body

< p >

(Voltar)

Determina o parágrafo dentro da estrutura, portando o texto irá receber configurações de um parágrafo, menos a indentação em seu início, isso será configurado no CSS.

< strong >

(Voltar)

Usado para determinar a importância de uma palavra dentro do texto, fazendo assim com que a palavra fique em negrito.

< em >

(Voltar)

Usado para destacar alguma palavra no texto, fazendo com que a mesa fique em itálico.

< a >

(Voltar)

Responsável por criar um link, podendo ser direcionado para outro site ou para alguma outra página ou parte do próprio site.

< ol >

(Voltar)

Tag mãe, usada junto com a tag filha < li > para criar listas ordenadas.

< ul >

(Voltar)

Tag mãe, usada junto com a tag filha < li > para criar listas não ordenadas.
